# 🚢 Titanic Outlier Detection 

📌 Overview

This project demonstrates a production-style **data preprocessing pipeline** for detecting and removing outliers in the Titanic dataset using the **Interquartile Range (IQR) method**. It includes data cleaning, statistical analysis, and visualization of distribution changes before and after outlier removal.

 🎯 Problem Statement

Real-world datasets contain noisy values and outliers that can degrade machine learning model performance. This project focuses on identifying and handling outliers in numerical features to improve data quality.
